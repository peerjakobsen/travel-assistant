# Command: trip-brief
Slash command: /travel:trip-brief

## Purpose
Generate a comprehensive pre-departure document the whole family can reference during the trip.

## Inputs
- None — uses the current trip context from the active planning session

## Instructions
> **Stub — implemented in Milestone 7**
>
> When implemented, this command will:
> 1. Gather all confirmed bookings and references from the current trip
> 2. Compile a complete trip document with all essential information
> 3. Output as a markdown file the family can print or share

## Output format
A structured document containing:
- Trip overview (dates, destination, family members)
- All confirmation numbers and booking references
- Accommodation address + check-in instructions
- Day-by-day itinerary
- Car rental pickup details
- Emergency contacts (local hospital, embassy, travel insurance)
- Packing list tailored to destination + activities + children's ages
- Pre-departure checklist (pet care, mail hold, etc.)
